### Como criar uma história criativa para crianças utilizando o ChatGPT e Canva

```markdown
# Criando texto e imagem utilizando ChatGPT e Canva

## 📒 Descrição
Foram utilizadas IA's Generativas para a criação de texto em formato de história e imagem ilustrativa.

## 🤖 Tecnologias Utilizadas
ChatGPT
Canva

## 🧐 Processo de Criação
Primeiramente, utilizando engenharia de prompt, foi solicitado ao ChatGPT que criasse uma história criativa para crianças
de 11 anos, utilizando como personagens um bem-te-vi, um pirata e uma bailarina em um cenário apocalíptico. Para ilustrar
a história, foi feito um resumo de quem são os personagens e o cenário, e foi solicitada à IA Canva a geração de imagens.

## 🚀 Resultados
O ChatGPT retornou com uma história bem interessante e emocionante, como se a história na verdade fosse a narração de um sonho.
As ilustrações geradas no Canva foram bem curiosas, inclusive em uma delas foi gerado um pirata com cabeça de bem-te-vi.
Foi escolhida uma das imagens que parecia se encaixar melhor à história.

## 💭 Reflexão
Para criar algo considerado "natty" com uma IA, é necessário saber escrever um bom prompt, pois dessa forma a IA conseguirá chegar o mais próximo possível de algo considerado real, ou que tenha sido feito por mãos humanas.
